## Jogo Da Velha
Jogue o clássico jogo da velha e teste suas habilidades contra uma IA!!\
A inteligência artificial aplica uma heurística baseada nos algorítimos [Minimax] e [Poda Alfa-beta].

### Como constuir o projeto
```sh
$ mkdir build
$ cd build 
$ cmake ..
$ make

```

## Jogar:

```sh
$ ./TicTacToe
```
